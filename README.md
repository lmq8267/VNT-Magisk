## [VNT](https://github.com/vnt-dev/vnt)的安卓面具模块

模块内置的二进制程序[vnt-cli](./vnt_Magisk/bin/vnt-cli)为`arm64`架构的，如需`armv7`架构请自行下载更换即可

安装之后，请修改`/data/adb/modules/vnt_magisk/config.toml`后重启插件。

可使用[在线配置文件生成器](https://lmq8267.github.io/VNT-Magisk/)进行生成，不懂的请详细阅读[官方文档](https://rustvnt.com/guide/gui.html)和[项目说明](https://github.com/vnt-dev/vnt/blob/1.2.x/vnt-cli/README.md)
